# README

**TronSave** 是一个用于租赁和管理 **TRON 能量与带宽**的市场。在进行 USDT（TRC‑20）转账等交易时，只需支付销毁 TRX 成本的一小部分；或通过出租你质押的 TRX 所产生的资源来赚取收益。

基于 TRON **质押 2.0** 构建。TRON 黑客松 S4 前三名 · S5 第一名 Builder。

{% hint style="info" %}
**初次到访？** 在 10 分钟内购买你的第一份能量 → [快速开始](getting-started/quickstart.md)。 **正在集成？** 直接前往[开发者快速开始](developers/quickstart.md)。
{% endhint %}

## 使用 TronSave 你可以做什么

* **购买资源**：在网站上、在 Telegram 上，或通过 API —— 支持 [Normal、Pending、Smart、ZapBuy、Auto Buy](concepts/order-types.md) 等订单类型。
* **集成**能量购买功能到你的 dApp 或后端，使用 [REST API](developers/api-reference/) 或 [SDK](developers/sdk.md)（TypeScript、Rust、Python、Java 和 PHP）。
* **出售 / 提供**来自你质押 TRX 的能量并[赚取 APY](concepts/pricing-and-apy.md)。
* **管理**订单、续期租赁，并通过[工具](guides/tools/)执行批量操作。
