# 扩展

扩展功能让你能从已经租赁或即将租赁的能量中获得更多价值。可用于：

* 最大化目标地址可接收的能量数量。
* 延长一块有效能量的时长，且不会有被回收的风险。
* 从同一个代理(委托)地址多次购买更多能量。

TronSave 提供两种方式来实现这一点：

<table>
<thead>
<tr><th>方式</th><th>作用</th></tr>
</thead>
<tbody>
<tr><td><a href="quick-extend.md">快速扩展</a></td><td>快速充值，只需几步即可延长一块有效能量。</td></tr>
<tr><td><a href="advance.md">高级</a></td><td>对从同一个代理(委托)地址重复购买进行精细控制。</td></tr>
</tbody>
</table>

{% hint style="info" %}
两种方式都建立在标准租赁流程之上。如果你还没有租赁过能量，请从[如何购买能量和带宽](../buy/README.md)开始。
</hint>
