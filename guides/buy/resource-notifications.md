# 资源通知

**Telegram 通知**让您在 Telegram 中即时收到关于订单和市场机会的提醒，从而无需频繁查看平台即可及时采取行动。

该功能涵盖市场的买卖双方：

* **买家通知（本页）** — 当租赁的资源订单即将被回收，或您的能量/带宽余额低于阈值时发出提醒。
* **卖家通知** — 当某个买单符合您的出售条件（Suggest Sell）时发出提醒，帮助您以理想价格更快地完成出售。请参阅 <!-- [NEEDS CONFIRMATION: target page for Seller "Suggest Sell" notifications does not yet exist under content/en] --> Suggest Sell 指南。

**资源通知**是买家侧的提醒：当已购买的能量或带宽订单接近其回收时间，或余额即将不足时，它们会通知您 — 以便您在服务中断之前进行延期、重新购买或充值。

进入 **BUYER**（买家）板块，点击 **NOTIFICATIONS**（通知）选项卡。

在提供的输入框中填入您的 Telegram ID，然后点击 **CONFIRM**（确认）。

## 通知类型

### 回收提醒

当已购买的能量或带宽订单即将到达其回收时间时通知您，以便您在资源被回收并变得不可用之前**延期或重新购买**该资源。

* **可配置的提前时间** — 选择您希望提前多久收到提醒（例如，在回收前 60 分钟）。

<figure><img src="https://1055070949-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fh0rlDWTk05Qu3lvB5Gve%2Fuploads%2F1pvMjvOljqrWhyrXD4cE%2Fimage.png?alt=media&#x26;token=c1bfcf25-6930-45c4-a935-25c3a91c7de2" alt=""><figcaption></figcaption></figure>

### 资源不足提醒

当您的能量或带宽降至您所设定的阈值以下时发出提醒，以便您在余额降到 `0` 并导致交易开始失败之前采取行动。

* **按资源类型控制** — 可为能量、带宽或两者同时启用提醒。
* **独立阈值** — 为每种资源类型设置不同的阈值。

<figure><img src="https://1055070949-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fh0rlDWTk05Qu3lvB5Gve%2Fuploads%2FJ0xCEX8P4rMhp3rOrcRN%2Fimage.png?alt=media&#x26;token=bfea405b-b567-4ef4-bef0-7cd51d41c9bf" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
回收提醒跟踪您的活跃租赁订单；资源不足提醒跟踪您地址上的实时能量/带宽余额。同时启用两者可获得全面的覆盖。请参阅 [能量与带宽](../../concepts/energy-and-bandwidth.md) 了解这些资源是如何被消耗的。
{% endhint %}
