# 连接钱包

在你能够于 [tronsave.io](https://tronsave.io) 上购买、出售或质押资源之前，你需要先连接一个 TRON 钱包。TronSave 支持三种连接方式：浏览器钱包、通过 API 密钥连接的内部账户，以及一系列流行的扩展和移动端钱包。

## 1. 浏览器钱包

1. 点击 **Connect** 按钮。
2. 选择 **Browser Wallet**。
3. 登录你的 TRON 钱包并批准连接。

{% hint style="info" %}
如果你的钱包连接到了错误的网络，请点击 **Wrong network** 并选择 **Switch** 以切换到正确的网络。
{% endhint %}

## 2. 内部账户（API 密钥）

如果你以编程方式与 TronSave 交互，你可以使用 API 密钥进行连接，而无需通过浏览器扩展登录。有关如何生成和使用 API 密钥的说明，请参阅[身份验证](../developers/authentication.md)。
