# 合约地址

TronSave 的 SDK 会与两个链上合约交互：**Forwarder** 和 **Relay Hub**。请使用与你目标网络相匹配的地址集——这两个环境使用不同的部署。

{% hint style="warning" %}
主网和 Nile 测试网的地址**不可**互换。在 Nile 测试网上使用主网地址（反之亦然）将会失败。关于这两个网络的差异，请参阅 [Environments](../environments.md)。
{% endhint %}
